Ejercicio #1

  • Agrega un div con una clase container
  • Dentro agrega cuatro parrafos
  • Cada parrafo debe tener una clase
  • Las clases deben ser:
    1. parrafo1
    2. parrafo2
    3. parrafo3
  • El parrafo1 debe tener la primera letra en mayúscula y un tamaño de 30px con la familia de fuente 'Times New Roman', Times, serif
  • El parrafo2 debe tener un color de fondo lightblue y un color de letra black
  • El parrafo3 el texto debe ser centrado y subrayado


Ejercicio #2

  • Crea una lista de enlaces con almenos 4 elementos
  • Deberás usar el selector de tipo para agregar un icono al principio de cada enlace
  • Puedes usar pseudo-elementos para agregarle los iconos (no te olvides de usar content)
  • Para agregar el icono debes usar el pseudo-elemento ::before
  • Por ultimo al colocar el mouse sobre los enlaces deberán cambiar el color del texto


Ejercicio #3

  • Crea un div con la clase contenido y dentro un parrafo
  • Dentro crea un div y dentro un parrafo
  • Ahora agregale un color de fondo lightcyan y el texto subrayado pero solamente al primer parrafo y desde la clase contenido
